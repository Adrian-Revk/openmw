Detailed Description

DispatchMap is a simple connection system that connects incomming signals with outgoing signals.

The signals can be connected one-to-one, many-to-one, one-to-many or many-to-many.

The dispatch map is completely system agnostic. It is a pure data structure and all signals are just integer indices. It does not delegate any actions, but used together with Dispatcher it can be used to build an event system.
