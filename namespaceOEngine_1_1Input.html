typedef boost::function<void(int,const void*) OEngine::Input::Action)

An Action defines the user defined action corresponding to a binding.

The first parameter is the action index that invoked this call. You can assign the same function to multiple actions, and this can help you keep track of which action was invoked.

The second parameter is an optional user-defined parameter, represented by a void pointer. In many cases it is practical to point this to temporaries (stack values), so make sure not to store permanent references to it unless you've planning for this on the calling side as well.
